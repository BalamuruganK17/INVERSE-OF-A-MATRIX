# INVERSE-OF-A-MATRIX
## Aim:
To write a python program to find the inverse of a matrix
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:
### Step1 : First import numpy as np
### Step 2: Declare A for the array of elements
### Step 3: Use inverse=np.linalg.inv(A) to get inverse of the given matrix
### Step 4: Finally print the inverse of the matrix

## Program:
```
#Program to find the inverse of a matrix.
#Developed by: BALAMURUGAN.K
#RegisterNumber: 25017932

import numpy as np
A=np.array([[2,1,1],[1,1,1],[1,-1,2]])
inverse=np.linalg.inv(A)
print(inverse)

```
